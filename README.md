## Запуск проекта

### 1. Клонирование

```bash
git clone https://github.com/rufathas/test-task
cd backend-test
```
### 2. Сборка и запуск проекта с тестовыми данными

```bash
make setup-project
```
### 3. Запуск тестов:
```bash
make run-unit-test
```

P.S
Для проверки API можно использовать коллекции Postman в проекте.
```text
Postman Collection.json
```
